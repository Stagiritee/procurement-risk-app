# Калькулятор риска закупки

Веб-приложение для расчёта риска по трём этапам закупки, определения общего уровня риска и формирования рекомендаций по его минимизации.

## Локальный запуск

```bash
npm install
npm run dev
```

## Production-сборка

```bash
npm run build
```

Готовый сайт после сборки находится в папке `dist/`.

## Как выложить сайт через Vercel

1. Создайте репозиторий на GitHub.
2. Загрузите в него этот проект.
3. Перейдите на `https://vercel.com/`.
4. Войдите через GitHub.
5. Нажмите `Add New` -> `Project`.
6. Выберите репозиторий с этим проектом.
7. Убедитесь, что Vercel определил настройки так:
   - Framework Preset: `Vite`
   - Build Command: `npm run build`
   - Output Directory: `dist`
8. Нажмите `Deploy`.
9. После завершения деплоя получите публичную ссылку на сайт.

## Проверка на телефоне

После деплоя:

1. Откройте ссылку сайта на телефоне.
2. Пройдите шаги 1-5.
3. Проверьте:
   - корректность кнопок и шкал;
   - отображение всех блоков;
   - скачивание PDF-отчёта.

## Что уже подготовлено

- проект собирается через `Vite`;
- сайт можно публиковать как статический фронтенд;
- есть `manifest.webmanifest` для более удобного открытия на мобильных устройствах.
